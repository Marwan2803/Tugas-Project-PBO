PetAppProject – Aplikasi Manajemen Hewan Peliharaan (Java OOP)

Deskripsi Proyek:
Proyek ini merupakan aplikasi Java sederhana yang dirancang untuk mengimplementasikan konsep-konsep OOP berikut:
- Class & Object
- Inheritance
- Polimorfisme
- Encapsulation
- Package
- Abstract Class
- Interface
- Generic & Collection

Struktur Folder:
PetAppProject/
├── README.txt
└── src/
    ├── Main.java
    ├── model/
    │   ├── Pet.java
    │   ├── Dog.java
    │   ├── Cat.java
    │   └── Trainable.java
    └── service/
        └── PetService.java

Cara Menjalankan:
1. Import proyek ke IDE Java (IntelliJ / Eclipse)
2. Pastikan folder `src/` dijadikan source folder
3. Jalankan file `Main.java`

Versi Java:
- Direkomendasikan menggunakan Java 8 ke atas
